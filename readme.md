# Data Timeseries 

## How to install
1. clone this repo
```
git clone https://github.com/bambangsy/timeseries-streamlit
```
2. On the file path, build virtual environment
```bash
python -m venv venv
```
3. Use the venv
```bash
venv\scripts\activate
```
4. install the libraries
```bash
pip install -r requirements.txt
```
5. start the app
```bash
streamlit run app.py
```
